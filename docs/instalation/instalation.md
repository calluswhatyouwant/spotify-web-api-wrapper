---

Here's how to install spotify-web-sdk library in your project:



### Using Yarn:

```sh
yarn add spotify-web-sdk
```

### Using NPM:

```sh
npm install spotify-web-sdk
```

### Using CDN:

```html
<script src="https://unpkg.com/spotify-web-sdk/build/dist/spotify-web-sdk.min.js"></script>
```
